DevOps a full (basic) process
Deployed for example/test.
GitHub - Create an account on GitHub (https://github.com/).
- Create a public (or private) Repository (https://github.com/carlodeba/carlodeba).
- Create an index.html file (the one you are reading now).
 DONE
DockerHub - Create an account on DockerHub (https://hub.docker.com/).
- Create a public (or private) Repository (https://hub.docker.com/r/carlodeba/carlodeba).
- Link the GitHub account.
- Setup autobuilds from the GitHub Repository.
- Create a Dockerfile.
- Commit index.html to GitHub, a new image will be built.
