Rancher2.X - Install a single node Rancher2.X.
- Install longhorn app for persistent storage.
- Install let's encrypt app.

GitHub - Create an account on GitHub (https://github.com/).
- Create a public (or private) Repository (https://github.com/carlodeba/carlodeba).
- Create an index.html file (the one you are reading now).
 DONE
DockerHub - Create an account on DockerHub (https://hub.docker.com/).
- Create a public (or private) Repository (https://hub.docker.com/r/carlodeba/carlodeba).
- Link the GitHub account.
- Setup autobuilds from the GitHub Repository.
- Create a Dockerfile.
- Commit index.html to GitHub, a new image will be built.
 DONE
Rancher 2.X - Create a k8s deployment with the fresh builded image.
- "kubectl create deployment test-site --image=carlodeba/carlodeba:latest".
- Add an ingress for the new workload with LE certificate.
- Now you can read this page!.

Rancher 2.X - Create a public (or private) Repository (https://github.com/carlodeba/charts).
- Add catalog to Rancher with URL: https://github.com/carlodeba/charts.git.
- Create a directory, it will be the APP in the catalog.
- Create and modify directories/files as in the example https://github.com/carlodeba/charts.
- Launch the APP from Rancher catalog.
- Create an ingress.
